KEY PROJECT

Complete SME IT Infrastructure

Design and deployment of a complete network and system infrastructure for a 30-user SME.

Windows Server 2019 Active Directory DNS/DHCP VLAN Fortinet/pfSense
Duration: 3 months | Status: Completed

Project Overview

Context & Challenge

A growing SME with 30 employees needed to migrate from a basic peer-to-peer network to a professional IT infrastructure that could support business growth, ensure data security, and provide reliable connectivity.

The company had no centralized user management, no network segmentation, and was experiencing frequent connectivity issues and security vulnerabilities.

Solution Implemented

  • Deployed Windows Server 2019 with Active Directory for centralized user and computer management
  • Implemented VLAN segmentation to separate users, management, and IT departments
  • Configured DNS and DHCP services for automatic IP management and name resolution
  • Set up pfSense firewall with security policies and traffic monitoring
  • Created Group Policy Objects (GPO) for standardized workstation configuration
  • Implemented backup and disaster recovery procedures
