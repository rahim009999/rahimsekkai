MONITORING

System Monitoring & Maintenance

Proactive monitoring of servers and network equipment to prevent failures.

Zabbix/Wazuh Windows/Linux SNMP Alerting SIEM
Duration: 4 months | Status: Completed

Project Overview

Context & Challenge

The company lacked visibility into its IT infrastructure, leading to unexpected downtimes and delayed responses to issues. There was no centralized monitoring system.

The goal was to implement a comprehensive monitoring solution that would provide real-time alerts, historical data for analysis, and proactive maintenance capabilities.

Solution Implemented

  • Deployed Wazuh SIEM on Ubuntu Server for security monitoring and log analysis
  • Configured Zabbix for infrastructure monitoring (servers, switches, firewalls)
  • Implemented SNMP monitoring on network devices
  • Set up alerting and notification rules for critical events
  • Created dashboards for real-time visualization of system health

Technical Details

  • Wazuh SIEM: Version 4.7 on Ubuntu 22.04
  • Zabbix: Version 6.4 with 50+ monitored hosts
  • SNMP: v2c and v3 on all network devices
  • Alerting: Email, Slack, and SMS notifications
  • Logs: 100GB/month analyzed

Skills Demonstrated

  • LAN/WAN monitoring with SNMP
  • Failure detection and alerting
  • Preventive maintenance planning
  • System alerts and incident response

Results Achieved

  • 60% reduction in unplanned downtime
  • Early detection of 90% of potential failures
  • Complete visibility across 50+ devices
